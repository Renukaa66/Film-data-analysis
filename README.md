# 🎬 Film Data Analysis 

## Overview
This project analyses film data loaded from 2 CSV files and 1 Excel workbook.

## Files
| File | Description |
|------|-------------|
| films.csv | CSV File 1 — 10 films with ratings and box office |
| reviews.csv | CSV File 2 — 14 critic reviews |
| film_assignment_FINAL.xlsx | Excel workbook with analysis + 4 charts |
| film_analysis.py | Python script version |

## Key Findings
- Avengers Endgame: Highest box office at $2,798M
- Get Out: Best ROI at 5,567% on a $4.5M budget
- Action genre dominates with 56% of total box office

## Tools Used
- Microsoft Excel (VLOOKUP, AVERAGEIF, Charts)
- Python (pandas, matplotlib, openpyxl)
